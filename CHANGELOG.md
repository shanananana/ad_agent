## [0.1.2]（2026-03-23）

### 对话工作台（`/chat.html`） +  创意素材生成(生图)（2026-03-23）

- **提示词外置**：主对话 / 调价 / 创意 / 意图识别 / 长期记忆 / 编排器 user 骨架 / 版位说明 / 规划层展示文案等统一放在 `src/main/resources/prompts/`，由 `ClasspathPromptLoader` 加载；Java 仅注入占位符与业务数据。（2026-03-23）
- 左侧 Tab：**对话 Agent**、**计划查看**、**效果查看**、**调价日志**、**素材生成**；与头部 **用户 ID**、侧栏 **当前计划** 联动。（2026-03-23）
- **素材生成**：选广告组、可选内容库与**版位**；文生图并可保存到全局素材、绑定到组（可插队首）；地址可带 `#tab=creative` 直达素材 Tab。（2026-03-23）
- **智能生成画面描述**：结合高 ROI 素材文案、内容库摘要、文本框草稿，由大模型生成描述；**版位**决定提示模板里注入的画幅与安全区说明；`PromptTemplate` 拼装；生成结果**按语义换行**写入描述框；按钮紧挨画面描述输入区右侧，便于发现。（2026-03-23）
- **计划树 → 效果**：`chat.html` 结构树中素材为可点击按钮，跳转 **效果查看** Tab，按素材 ID 筛表明细，汇总卡片为筛选后数据；可「显示全部明细」；地址栏支持 `#tab=effects&creativeId=`。（2026-03-23）
- **README**：对话工作台 **素材生成** 配图使用 `docs/images/chat-material-gen.png`（由原 `text_2_png.png` 重命名）；**计划查看** 暂为文字说明。（2026-03-20 / 2026-03-23）
