## [0.1.1] - 2026-03-20

### 2026-03-20 · 效果看板（bid-strategy 增强）

- 投放效果 **按日趋势图**（整计划 / 按明细行），指标与天数可选；**调价成功日竖线**（仅时间参考）。
- 明细表 **迷你折线**、**转化 / 创意** 列、**首列冻结**（日期、人群）、消耗/CTR/ROI **数字格式**（2 位 / 4 位）。
- 接口：`performance-series`、`performance-sparklines`；`CampaignPerformanceSeriesService`。

### 2026-03-20 · 自动调价 B×α

- 定时任务、计划目录 `data/bid/campaigns/<计划ID>/` 下出价模型/快照/系数、LLM 调价与护栏、任务日志。
- `bid-strategy.html` 与 `BidStrategyController`：计划列表/详情、效果汇总与样本生成、概览与调价操作。

### 2026-03-18 · 对话体验与隐私

- **意图识别**：注入「最近对话」上下文，支持短句确认（如「好的」「需要」）。
- **隐私清除**：自然语言触发后服务端删除长期记忆/聊天记录文件；流式响应可下发表页刷新；`chat.html` 配合清本地状态。
- **对话页**：用户 ID 必填校验与提示；效果数据筛选空结果修复；配置与密钥示例说明更新。

### 2026-03-17 · 项目初始化

- **对话式投放 Agent**：Spring Boot + Spring AI（千问），感知 / 规划 / 执行分层，工具链（查数据、效果、改计划等）。
- **短期记忆**：当前会话内多轮对话上下文，供连贯理解与追问。
- **长期记忆**：跨会话用户偏好/习惯，按用户写入本地 JSON，可检索注入后续轮次。
- **聊天记录**：多会话管理、按用户索引、会话历史持久化；与短期记忆配合使用。
- **流式回复**：SSE、思考过程展示；多用户数据目录隔离。
- **工程**：`.gitignore` 忽略运行时数据；README 含快速开始与生产存储建议。
