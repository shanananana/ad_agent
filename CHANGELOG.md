# 更新日志

## [0.1.2]

### 对话工作台（`/chat.html`） +  创意素材生成(生图)

- 左侧 Tab：**对话 Agent**、**计划查看**、**效果查看**、**调价日志**、**素材生成**；与头部 **用户 ID**、侧栏 **当前计划** 联动。
- **素材生成**：选广告组、可选内容库与**版位**；文生图并可保存到全局素材、绑定到组（可插队首）；地址可带 `#tab=creative` 直达素材 Tab。
- **智能生成画面描述**：结合高 ROI 素材文案、内容库摘要、文本框草稿，由大模型生成描述；**版位**决定提示模板里注入的画幅与安全区说明；`PromptTemplate` 拼装；生成结果**按语义换行**写入描述框；按钮紧挨画面描述输入区右侧，便于发现。
- **计划树 → 效果**：`chat.html` 结构树中素材为可点击按钮，跳转 **效果查看** Tab，按素材 ID 筛表明细，汇总卡片为筛选后数据；可「显示全部明细」；地址栏支持 `#tab=effects&creativeId=`。
- **README**：增加对话工作台截图说明（`docs/images/text_2_png.png`）。
