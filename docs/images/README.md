# README 配图

本目录存放 `README.md` 中引用的截图，便于在 GitHub 上展示 **自动调价助手** 界面。

| 文件 | 说明 |
|------|------|
| `bid-strategy-performance-detail.png` | 计划详情：投放效果汇总 + 多维度效果明细 |
| `bid-strategy-b-alpha-grid.png` | 本计划 B×α 调价网格与操作按钮 |
| `bid-strategy-job-log.png` | 本计划调价任务日志（涨跌概要 + LLM 说明） |

更新截图时 **覆盖同名 `bid-strategy-*.png` 即可**。

若编辑器导出为 `img.png`、`img_1.png`、`img_2.png`，请按上表含义重命名（本次已按体积大致对应：最大 → 效果明细页，中等 → B×α 网格，较小 → 任务日志）。若与页面实际不符，对调三个文件名即可。

根目录 `README.md` 中三张图使用 HTML `<details>` / `<summary>` 包裹：**默认只显示标题，点击再展开图片**（GitHub 网页端支持；部分本地预览器可能表现不一致）。
